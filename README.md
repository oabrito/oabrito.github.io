# Este é o terceiro exercicio de git do curso da OneBitCode


## O Exercicio é realizado pro partes


* ### 1° É necessario criar o repositorio remoto no GitHub

* ### 2° Criar o gepositorio local com
```bash
git init
```

* ### 3° Adicionar os arquivos anter de commitalos

```bash
git add nome-do-arquivo
```

* ### 4° Realizo o commit dos aquivos

```bash
git commit -m "mensagem explicando as alterações"
```

* ### 5° Adiciono o repositorio remoto na maquina local

```bash
git remote add origin ssh-do-repositorio-remoto
```

* ### 6° Enviando os commits da branch atual para o repositorio remoto  'nomeado como origin'

```bash
git push repositorio-remoto branch-atual
```
